# Multi Tool Calling Agent

An AI-powered agent built using Streamlit, Groq LLM, and Python that can answer user queries by calling external tools.

## Features

- Weather Information
- Stock Price Lookup
- Command Execution
- Tool Calling Workflow
- Streamlit Chat Interface

## Tech Stack

- Python
- Streamlit
- Groq API
- OpenAI SDK
- Requests
- Python-dotenv
